Bandwidth, Latency and Identifier

Bandwidth, latency and identifier are not the intrinsic properties of a path. So these properties is assigned when the path is build from a graph and can be altered thereafter.

Constructor

Build path with a list of node:

    Path(node_list, bandwidth, latency)

Build path with existing Path instance:

    Path(path)

Property

  • Length get
  • Source get
  • Destination get
  • TailLink get
  • HeadLink get
  • node_list set
