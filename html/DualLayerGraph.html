Constructor

Build(network)
Build(layer_size, seed, link_capacity)

layer_size: the graph size of each layer. If layer_size is scalar, the two layer have the same size.

seed: seed of random number generator for creating graph.

link_capacity: the link capacity of each layer. If this argument is not specified, the link capcity is set to 1. If link_capcity is a scalar, the link capacity of the two layer is the same.
